# Software Planning Project
Project for the course "Progettazione del Software e dei Sistemi Informativi" - UniTS.

## Description
The project consists in an arXiv paper visualizer, where papers' abstracts are first converted into a semantic embedding, compressed through a trained Autoencoder and then used for k-NN to find the closest papers.
